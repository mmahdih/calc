# 🧮 Calculator App

A modern React calculator with gradient UI.

## Quick Start

```bash
# Clone or create React app
npx create-react-app calculator-app
cd calculator-app

# Replace src/App.js with the calculator code
# Replace src/App.css with the calculator styles

# Run
npm start
# The app opens at http://localhost:3000
```

# Features

* Basic math operations (+, −, ×, ÷, %)
* Decimal support
* Clear & Backspace
* Mobile responsive
* No dependencies

# Tech Stack
* React + Hooks
* CSS3 Gradients & Grid
